# Referencias

Permite realizar la consulta y edición de los datos de referencias de una persona (Natural o Jurídica). 

Las referencias varían dependiendo del tipo de persona que sea el cliente. Al seleccionar un tipo de referencia, la sección se expande para mostrar los datos existentes y habilita las opciones para gestionar los registros actuales o agregar nuevos. Para guardar los cambios realizados, el sistema solicitará las credenciales que autorizan la modificación de los datos del cliente.

![Administrar Referencias](../assets/images/administrar_persona_-_referencias_.png)

En la sección de referencias se puede observar de forma general:
* La cantidad de registros existentes por cada referencia.
* La cantidad mínima y máxima permitida por cada referencia (excepto las referencias SIAFI, que son opcionales). 

La cantidad de registros obligatorios se determina en base al valor indicado en el campo **Registros Mínimos**. Del mismo modo, el campo **Registros Máximos** limita el ingreso de nuevas referencias una vez alcanzado el tope.
