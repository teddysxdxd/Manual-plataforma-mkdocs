# Cónyuge

Permite realizar la consulta de los datos del cónyuge de una persona (Natural). 

Esta sección se visualiza únicamente cuando el estado civil de la persona está configurado para requerir datos de cónyuge. Para consultar esta información, se utiliza el botón **Administrar Cónyuge**, el cual redirecciona al administrador de personas. Desde allí se puede navegar por toda la información de la persona relacionada y, utilizando la misma opción, retornar al cliente principal.
