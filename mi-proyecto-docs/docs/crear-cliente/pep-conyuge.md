# PEP Cónyuge - Persona Natural

Permite registrar la información del cónyuge de la persona PEP, esta sección se muestra solo cuando la persona PEP tiene un estado civil que requiera datos de cónyuge. La captura de datos del cónyuge está dividida en varios pasos, para continuar al siguiente paso se debe completar la información requerida en cada sección. 

* [Información General – PEP Cónyuge](informacion-general-pep-conyuge.md)
* [Dirección Residencia – PEP Cónyuge](direccion-residencia-pep-conyuge.md)
* [Contacto – PEP Cónyuge](contacto-pep-conyuge.md)
* [Referencias Laborales – PEP Cónyuge](referencias-laborales-pep-conyuge.md)
* [Datos Negocio – PEP Cónyuge](datos-negocio-pep-conyuge.md)
