# Código de cliente

Permite realizar la búsqueda de una persona por medio del código de cliente con el que está creado en el sistema. Este campo es específico; no se debe indicar más datos de combinación en la búsqueda. 

| Campo | Descripción | Condición |
| :--- | :--- | :--- |
| **Código de cliente** | Indicar el código de cliente específico a buscar. | Opcional |
